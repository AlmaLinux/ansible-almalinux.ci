# almalinux.ci.nvidia_cuda

An Ansible role that installs the Nvidia CUDA toolkit.


## Role Variables

The role variables and their default values are listed below:

* `nvidia_cuda_version: ''` - (**required**) a CUDA toolkit version (e.g. 11-6).
